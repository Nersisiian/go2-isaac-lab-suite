# 🤝 Contributing to GO2 Isaac Lab Suite

Спасибо за интерес к проекту! Мы рады любым улучшениям: от исправления опечаток до новых алгоритмов управления.

## 🧭 Как помочь

- 🐛 **Сообщить об ошибке** – создайте Issue с меткой `bug`.
- 💡 **Предложить идею** – Issue с меткой `enhancement`.
- 🔧 **Исправить код** – создайте Pull Request.
- 📖 **Улучшить документацию** – PR с меткой `documentation`.

## 🚀 Процесс разработки

1. **Форкните репозиторий** → кнопка `Fork` на GitHub.
2. **Склонируйте форк локально**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/go2-isaac-lab-suite.git
   cd go2-isaac-lab-suite
3. Создайте ветку (от main):
git checkout -b feature/your-feature-name
4. Внесите изменения и закоммитьте:
git add .
git commit -m "Add: краткое описание изменений"
5. Запушьте в свой форк:
git push origin feature/your-feature-name
6. Создайте Pull Request на GitHub → выберите ветку main оригинального репозитория.

✅ Требования к PR
Все статус-чеки (build, lint, security) должны быть зелёными.

Если добавляете новый функционал – добавьте хотя бы один тест (в tests/).

Код должен проходить flake8 (можно проверить локально).

Название PR и коммитов – на русском или английском, краткое и понятное.

🔧 Локальная проверка
# Установите зависимости
pip install -r requirements.txt
pip install -e .

# Запустите линтер
flake8 go2_extreme/

# Запустите тесты (если есть)
pytest tests/
📞 Вопросы
Если нужна помощь – создайте Issue с меткой question. Мы постараемся ответить в течение 3 дней.

Спасибо, что помогаете делать проект лучше! 🦾
